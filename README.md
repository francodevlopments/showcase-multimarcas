# Urban Select

Showcase fictício de e-commerce multimarcas de moda urbana, criado com React, TypeScript, Vite e Tailwind CSS.

## Rodar localmente

```bash
pnpm install
pnpm dev
```

## Build de produção

```bash
pnpm build
```

## Onde editar

- Nome, logo, WhatsApp, Instagram, endereço e horários: `src/data/site.ts`
- Produtos, marcas, preços, tamanhos, cores e imagens: `src/data/products.ts`
- Cores do tema: `tailwind.config.ts`
- Estilos globais e botões: `src/index.css`
